> RabbitMQ Publisher Subscriber  


## Install

```sh
$ npm install --save rabbitmq-pubsub
```


## Usage

```js
var rabbitmqPubsub = require('rabbitmq-pubsub');

rabbitmqPubsub('Rainbow');
```
